# 班級經營 - 家長回饋與電子獎狀生成系統使用說明

本專案旨在提供一個美觀的網頁介面，讓家長填寫回饋（姓名、Email、回饋文字），並在送出後**自動將資料紀錄至 Google 試算表**，同時**動態生成客製化的電子獎狀**供家長下載。

---

## 📁 專案檔案結構與存檔路徑

所有檔案皆存放在此資料夾內：
`d:\Antigravity\05_班級經營\06_給家長的回饋獎狀`

1. **`index.html`**：網頁主程式（表單與獎狀繪製系統）。
2. **`google_script.js`**：Google Apps Script 程式碼（用來串接 Google 試算表）。
3. **`blank_cert.png`** (⚠️ **需要您提供**)：
   - 請從您的 Canva 專案中，**將原本寫著名字「尤芸鳳」的文字框刪除（保留空白）**。
   - 將該空白獎狀下載為 **PNG** 格式，並命名為 `blank_cert.png`。
   - 將此檔案儲存到本資料夾中（與 `index.html` 同一個資料夾）。
   - *備註：若尚未放入此檔案，網頁程式會自動改以「高質感向量獎狀」做為備份顯示，以確保功能正常運作！*

## 🛠️ 第一步：設定 Google 試算表（收集家長回饋）

為了讓填表資料能自動寫入您的 Google 試算表，請按照以下步驟設定：

1. 開啟您的 **[Google 雲端硬碟](https://drive.google.com/)**，點擊左上角「新增」->「Google 試算表」。
2. 建立新試算表後，點擊上方功能表的 **「擴充功能」** -> **「Apps Script」**。
3. 清空 Apps Script 編輯器內原本的所有預設程式碼，並開啟本專案資料夾下的 [google_script.js](file:///d:/Antigravity/05_班級經營/06_給家長的回饋獎狀/google_script.js)，**複製全部內容並貼上**。
4. 按下 Ctrl + S（或點擊磁碟圖示）**儲存專案**。
5. 點擊右上角的 **「部署」** 按鈕 -> 選擇 **「新增部署」**。
6. 在新增部署視窗中：
   - 點擊左側「選取類型」旁的齒輪圖示，選擇 **「網頁應用程式」** (Web App)。
   - **說明**：輸入 `家長回饋收集`（或任意名稱）。
   - **專案指派給**：選擇您的 Google 帳號。
   - **誰能存取**：**務必改選為「所有人」 (Anyone)**（這樣家長填表時，程式才能成功寫入您的試算表）。
7. 點擊「部署」按鈕。此時 Google 會跳出權限請求，請點擊「授予存取權」，選擇您的 Google 帳號，並點擊「進階」->「前往『未命名專案』（安全）」來授權。
8. 部署完成後，畫面上會顯示 **「網頁應用程式 URL」**（網址開頭為 `https://script.google.com/macros/s/...`），**請將這個網址複製下來**。

## 🔗 第二步：將試算表網址填入網頁程式碼

1. 使用文字編輯器（如 Notepad++、VS Code 或記事本）開啟本專案資料夾下的 [index.html](file:///d:/Antigravity/05_班級經營/06_給家長的回饋獎狀/index.html)。
2. 尋找大約在 **第 351 行** 的 `CONFIG` 設定區：
   ```javascript
   // ==========================================
   // ⚙️ 設定區
   // ==========================================
   const CONFIG = {
     // ⚠️ 請在此處填入您部署 Google Apps Script 後產生的網頁應用程式 URL
     googleAppScriptUrl: "在這裡貼上您剛剛複製的網頁應用程式 URL",
     
     // 空白獎狀的圖片檔名 (請將空白圖片放在與 index.html 相同的資料夾)
     blankCertImage: "blank_cert.png"
   };
   ```
3. 將您剛剛複製的 `https://script.google.com/...` 網址貼到 `googleAppScriptUrl` 的雙引號中，然後儲存檔案！

## 🌐 第三步：如何開啟與部署網頁

### 本地使用與測試
在您的電腦上，直接**雙擊連點 [index.html](file:///d:/Antigravity/05_班級經營/06_給家長的回饋獎狀/index.html) 檔案**，即可在瀏覽器（Chrome、Edge 等）中打開此網頁並進行填表與下載測試。

### 分享給家長使用 (部署到網路上)
如果您希望家長在手機或自己電腦上填寫，您需要將 `index.html` 與 `blank_cert.png` 部署到免費的網頁空間。
推薦以下兩種極度簡單的免費部署方式：

#### 方案 A：使用 GitHub Pages (推薦，完全免費且最穩定)
1. 註冊並登入 [GitHub](https://github.com/)。
2. 建立一個新的公開 Repository（儲存庫），例如命名為 `parent-award`。
3. 將本資料夾內的 `index.html` 與您的 `blank_cert.png` 上傳至該儲存庫。
4. 到該儲存庫的 **Settings** -> **Pages**。
5. 在 **Build and deployment** 下的 Source 選擇 `Deploy from a branch`，Branch 選擇 `main` (或 `master`) 的 `/ (root)`，然後點擊 Save。
6. 大約 1-2 分鐘後重新整理頁面，最上方就會出現您的專屬網頁網址（如：`https://您的帳號.github.io/parent-award/`），複製該網址即可發送給家長！

#### 方案 B：使用 Vercel 或 Netlify (一鍵拖放部署)
1. 開啟 [Vercel 拖放上傳頁面](https://vercel.com/import/project) 或 Netlify。
2. 將包含 `index.html` 與 `blank_cert.png` 的資料夾整包拖曳上傳。
3. 系統會自動為您生成一個公開的網址，複製該網址即可。

## ✨ 系統亮點功能：獎狀姓名微調面板
當家長提交表單並進入獎狀生成畫面時，由於每張 Canva 空白獎狀的解析度或名字位置可能略有不同，網頁下方貼心設有 **「🛠️ 獎狀姓名位置微調」** 收折面板。家長（或您在測試時）可以透過滑桿，即時調整名字的：
* **垂直位置 (Y-axis)**
* **水平位置 (X-axis)**
* **字型大小**
* **字體顏色**

調整完後，點擊「下載獎狀」即可下載與畫面預覽一模一樣的完美電子獎狀！
